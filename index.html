Introdução ao Amigo One

Atualizado em 28/05/2025 Release v2.4.0

O Amigo One é a plataforma central de gestão financeira e relacionamento com clientes da Amigo. Este documento descreve sua arquitetura, funcionalidades principais e diretrizes de uso.

O que é o Amigo One?

Uma plataforma integrada que unifica o ciclo de vida completo do cliente — desde a aquisição até a fidelização — com módulos de pagamentos, relatórios e gestão de usuários.

Principais funcionalidades

  • Processamento de pagamentos em tempo real
  • Dashboard analítico com KPIs customizáveis
  • Gestão completa de perfis de usuários
  • Integração com sistemas externos via API REST
  • Relatórios automáticos por período

Arquitetura geral

A plataforma é construída em arquitetura de microserviços, com frontend em React, backend em Node.js e banco de dados PostgreSQL. A comunicação entre serviços é feita via mensageria assíncrona.
